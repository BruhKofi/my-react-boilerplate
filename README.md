# React boilerplate
## Tasks

### Bundle dev/production

**Make the dev build with Hot Module Replacement on `localhost:3000`**

```
yarn start
```

**Make the production build**

```
yarn build
```

### Deployment

**Deploy to Surge.sh**

Replace YOUR_DOMAIN with your desired domain name.

```
yarn deploy
```


**Deploy to Github pages**

```
yarn deploy-gh
```
